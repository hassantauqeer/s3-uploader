# React Basic Example

Simple single-file upload example using S3Up with mock provider.

## Features

- Single file upload with `useUpload` hook
- Mock provider (no backend required)
- File validation (size and type)
- Real-time progress tracking
- Error handling
- Beautiful UI

## Running

```bash
pnpm install
pnpm dev
```

Open http://localhost:5173

## What's Demonstrated

- Using `useUpload` hook for simple uploads
- Mock provider for testing without backend
- File validation configuration
- Progress tracking
- Success/error states
- Reset functionality
